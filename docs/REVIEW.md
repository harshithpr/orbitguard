# Where To Get Feedback

After OrbitGuard is on GitHub and Vercel, ask for targeted feedback.

## Best Places

- School STEM club
- Physics teacher
- Computer science teacher
- Science fair mentor
- Local university aerospace or physics outreach
- Reddit `r/aerospace`
- Reddit `r/space`
- Reddit `r/webdev`
- GitHub Issues
- LinkedIn post

## Feedback Questions

```text
Is the sustainability scoring understandable?
Are the simulator inputs realistic for a satellite launch?
Does the dashboard explain orbital debris clearly?
What would make the API useful for students or researchers?
What should I improve before presenting this in a portfolio?
```

Add these to the README once people respond:

- Number of reviewers
